Adafruit-MPL3115A2-PCB
======================

PCB files for Adafruit MPL3115A2


This is the PCB for the Adafruit 5V ready MPL3115A2 Barometric 
Pressure/Temperature/Altitude Sensor

Format is EagleCAD schematic and board layout
